ğŸ“Œ What is a Discord Fingerprint?

A Discord fingerprint is a unique identifier generated by Discord when a client connects to their API.
It is returned by the /api/v9/experiments endpoint and is used internally by Discord for:

Client identification

Feature flagging & experiments

Anti-abuse and telemetry systems

Fingerprints are typically generated per session and can vary depending on:

TLS handshake

Client identifier (browser version)

IP / proxy

Request behavior

This project demonstrates how fingerprints can be fetched programmatically using a TLS-fingerprinted HTTP client.
